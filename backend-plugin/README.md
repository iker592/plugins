# Backend Plugin

Comprehensive backend development plugin for Python projects with uv package manager.

## Features

- **python-uv-backend Skill**: Complete workflow for Python + uv development
  - Verification scripts (ruff, mypy, pytest)
  - Pre-commit hooks setup
  - Configuration templates
  - CI/CD templates
  - Best practices and troubleshooting

## Installation

```
https://github.com/iker592/plugins/backend-plugin
```

Or clone locally:

```bash
git clone https://github.com/iker592/plugins.git
# Add backend-plugin path to Claude Code settings
```

## Usage

### python-uv-backend Skill

Automatically activates when working with uv projects (`uv.lock`) or Python projects.

**Manual triggering**:
- "Set up a new Python + uv project"
- "Verify my Python code"
- "Configure pre-commit hooks"
- "Run all checks on my backend code"

## Components

- **Skills**: `python-uv-backend/`
